# International Bond

## Total International Bond Index

**Note**: This fund is not necessary in a US-based, Boglehead Three-fund portfolio but we are covering it because it is almost
always included in [All-In-One Funds](../portfolio/all-in-one).

### Overview

The International Bond Index provides broad exposure to non-US investment-grade, fixed-rate debt markets.

Funds typically track the **Bloomberg Global Aggregate ex-USD Float Adjusted RIC Capped Index (Hedged)**.

Its average duration is currently ~ 6 years.
